backdoor::Data Browser

Here you can browse the 'raw' data that Rita consumes, produces and archives. In addition (coming soon) we will have a more 'programmatic' query tool that will allow you to select specifically the data you are interested in, e.g. within a particular date range and geographic region.

IMPORTANT NOTE: these browser pages are NOT intended to be the channel by which you access large volumes of data, or try to emulate real-time access by repeatedly polling these pages. Rita is fundamentally architected to be a real-time data collection, analysis and propagation platform and we will provide 'push' data feeds to support that style of use in due course.
